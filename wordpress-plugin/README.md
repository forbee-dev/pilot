# MicroFE WordPress Plugin

WordPress plugin that integrates React components from MicroFE into Gutenberg blocks.

## Installation

1. Copy this plugin directory to your WordPress `wp-content/plugins/` directory
2. Activate the plugin in WordPress admin
3. Configure the MicroFE API URL in Settings > MicroFE

## Development

```bash
npm install
npm run build
```

## Features

- Gutenberg block for inserting React components
- Dynamic props configuration based on component schema
- SSR preview in editor
- Automatic hydration on frontend
- SEO-friendly server-side rendering
